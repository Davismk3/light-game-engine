# light-game-engine
A very lightweight C++ game engine/template for game development.

The primary purpose of this project is to generalize and reduce overhead from OpenGL, GLFW, and ENet into easy-to-understand modules for game development. This is in active development alongside a Python-to-C++ rewrite of another project. I figured I would take the time to design a proper template for myself, since starting new projects has been very difficult for me otherwise---having to return to OpenGL tutorials, etc. after forgetting how to set up libraries. Since this contains no source code for my projects that will be using this template, I do not mind sharing it and taking feadback, suggestions, or even assistance.
